Assignment 6 Problem 1. [15 Marks for correctness. File: cycle.c]

Suppose you are given an array A of n integers. Each integer in the array gives the number of positions to jump forward or backward from the current position. For example, if the current position is 2 and A[2] is 3, your next position will be 5. If A[5] is -4, then your next position will be 1. While calcuating the next position, you should consider the followings:

  • The next position must be modulo n, since i+A[i] can be greater than n.
  • If the next position becomes less than zero, you have to add n to it.
  • A cycle is possible if some sequence of jumps takes us to a previsously visited position.

Your task is to implement the function int count_cycles(int A[], int n); that will return the number of cycles in A. Your solution must assume the followings:

  • There will be at least one element in A
  • The jump starts with index zero in A.
  • There will be no jump of size 0 in A.
  • Absolute value of a jump will be between 1 and n. i.e., -n ≤ A[i] ≤ -1 or 1 ≤ A[i] ≤ n.
  • You can mutate A in any way you like.
  • There will be at least one cycle in A.
  • You can safely assume that a cycle is found whenever you jump into a previously visited position.

Public test samples:

int A[] = {3, 8, 4, 2, 7, 3, 1, -5, 5, -5};
assert(count_cycles(A, 10) == 3);
int B[] = {1, 1, 1};
assert(count_cycles(B, 3) == 1);

Assignment 6 Problem 5. [15 Marks for correctness, File: sudoku.c]

In this problem you will implement a function to check if a Sudoku board is valid or invalid. Sudoku is a popular number placing puzzle game. The aim is to fill a 9X9 grid of numbers without violating the following constraints:

  1. Each cell in the 9X9 grid can have a digit between 1 and 9
  2. Each row contains the same number only once
  3. Each column contains the same number only once
  4. Each 3X3 box with upper-left corner at locations (0,0), (0,3), (0,6), (3,0), (3,3), (3,6), (6,0), (6,3) and (6,6) contains the same number only once.

A Sudoku board can be partially or completely filled with digits. A board is considered to be valid if it conforms to the above rules. We can represent a Sudoku board by using an array of 81 characters, where the first 9 characters represent the first row, the next 9 characters represt the second row and so on. Implement the function
bool is_valid(char *board);
that will take an 81 character sudoku board, and will return true if the board is valid, i.e., conforms to the above rules. Your function should be able to handle empty cells in the board. We will use the character '.' to represent an empty cell.

Public test samples:
